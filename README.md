#### information-manege



#### Installation

Node.js 0.6 or higher is required.

```shell

$ git clone https://github.com/NNYrose/info.git

$ cd info 

$ npm install

$ node app.js

```

#### Config

`config.js` 中设置数据库类型 `( mysql or sqlite3)`;

```javascript
const DBNAME = 'sqlite3'; // mysql
```
